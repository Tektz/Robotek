# Robotek
Remote controlled Robot Arm with 360 rotating base
A multipurpose robort arm
Design and development by sTek web: www.stekdigital.com IG: @stekdigital

**Requirements:** Tools
1.	Stepper motor
2.	Servo motor 
3.	Motor driver/ ULN2003APG
4.	Arduino UNO
5.	IR receiver
6.	IR remote control
7.	Jumper wires
